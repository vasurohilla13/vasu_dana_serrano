# Skill: xero

## Description
Access Xero accounting records.

## Base URL
`$XERO_API_URL` (e.g., http://localhost:8080/xero)

## Endpoints

### GET /xero/invoices
Returns invoices for linked organisation.

### GET /xero/contacts
Returns contacts.

## Notes
- Dana does not have an active Xero organisation. This service is not configured for personal use.
